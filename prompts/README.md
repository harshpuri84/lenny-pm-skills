# Standalone Prompts

These prompts contain the same curated Lenny's Podcast wisdom as the Claude Code skills, packaged for use in any LLM.

## How to Use

1. Copy the contents of any prompt file below
2. Paste it into ChatGPT, Claude, Gemini, or any LLM
3. Follow the interactive workflow
